NAME

ucaresystem --- All-in-one system update and maintenance tool  

SYNOPSIS

ucaresystem [options ]  

DESCRIPTION

ucaresystem. is All-in-one system update and maintenance tool. It is written as an augmented tool to the distro specifics tools, and it doesn't try to replace them. It will automate the following:  

Basic Maintenance Tasks

Update the list of available packages. Download and install all the available updates. Removes obsolete packages Removes orphan packets. Deletes package configuration files from packages that have been uninstalled by you. Checks if there are older Linux kernels on the system and removes them. Check kernel removal for details.
 

This is the default task is no options is given.
